ToExcel
=


## 介绍

ToExcel 是使用 **superagent** 、 **cheerio** 、 **async** 、**json2xls** 、**electron** 搭建的抓取网页数据并导出为Excel的应用

## 安装部署

安装依赖

```bash
npm install
```

运行应用

```bash
npm start
```

打包生成exe

```bash
npm run package
```
